# AWS Static Website Hosting

## Project Overview
My portfolio website hosted on AWS using S3 and CloudFront.

## Architecture
User → CloudFront → S3 Bucket

## Services Used
- AWS S3
- AWS CloudFront  
- Terraform

## Deployment Steps
1. terraform init
2. terraform plan
3. terraform apply
4. terraform destroy
